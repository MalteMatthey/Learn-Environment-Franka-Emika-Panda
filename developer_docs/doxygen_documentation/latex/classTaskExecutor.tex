\doxysubsection{Detailed Description}
Executes tasks and manages their execution state. 

The \mbox{\hyperlink{classTaskExecutor}{Task\+Executor}} class is responsible for executing tasks and managing their execution state. It interacts with \mbox{\hyperlink{classScriptWorker}{Script\+Worker}} objects to perform the actual execution of tasks.
