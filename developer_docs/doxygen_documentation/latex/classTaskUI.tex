\doxysubsection{Detailed Description}
Manages the user interface for tasks and subtasks. 

The \mbox{\hyperlink{classTaskUI}{Task\+UI}} class is responsible for initializing and updating the UI elements related to tasks and subtasks. It interacts with the \mbox{\hyperlink{classTaskManager}{Task\+Manager}} and \mbox{\hyperlink{classSidebar}{Sidebar}} to provide a cohesive user experience.
